# Koboweb API

> Backend API For Koboweb

## Usage

Rename "env.example" to ".env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run watch

# Run in prod mode
npm start
```
